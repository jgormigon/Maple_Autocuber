# Tools Directory

This directory contains helper scripts and utilities for the Maple Autocuber bot.

## Files

- **`crop_region_tuner.py`** - Interactive GUI tool to tune crop region offsets in real-time
- **`find_crop_region.py`** - Script to find and visualize crop regions
- **`test_crop_ocr.py`** - Test script for OCR on crop regions
- **`positionfinder.py`** - Utility to find positions on screen
- **`autoclicker.py`** - Standalone autoclicker utility
- **`pickup.py`** - Standalone pickup utility

## Usage

These tools are separate utilities and are not required for the main bot to function. They can be run independently for testing, debugging, or configuration purposes.

### Running a tool

```bash
python tools/crop_region_tuner.py
```

Or from the tools directory:
```bash
cd tools
python crop_region_tuner.py
```
